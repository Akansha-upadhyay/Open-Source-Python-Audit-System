#Project Description
This project presents an analysis of Python as an open-source software. It discusses its background, core principles, licensing, and how it is used in Linux environments. The project also includes shell scripts to demonstrate practical implementation and understanding.

#Selected Software
Python is a popular open-source programming language known for its simplicity, readability, and versatility.

##Included Scripts

#1. script1.sh – System Information Report
Displays key system details such as user name, kernel version, system uptime, date, and Python-related information.

#2. script2.sh – Python Checker
Checks whether Python is installed on the system and shows its version along with basic details.

#3. script3.sh – Directory Audit Tool
Reviews important system directories and reports their permissions along with disk usage.

#4. script4.sh – Log Analyzer
Analyzes a log file to count occurrences of a specific keyword (such as "error") and displays the findings.

#5. script5.sh – Manifesto Creator
Generates a personalized open-source statement based on user input.
